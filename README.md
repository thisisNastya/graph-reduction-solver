# Универсальный калькулятор направленных графов

Этот скрипт представляет собой консольную утилиту на Python для расчета параметров направленных графов. Программа автоматически анализирует структуру графа и предлагает подходящие математические методы для нахождения итоговых значений всех вершин (узлов).

## Поддерживаемые типы графов

Программа умеет работать с любыми направленными взвешенными графами:
* **Ацикличные графы** (классические сети СПУ, без петель и обратных связей).
* **Циклические графы** (сложные сети, содержащие петли самовозврата и обратные связи между узлами).

*Программа включает встроенный алгоритм (DFS), который автоматически проверяет граф на наличие циклов перед началом расчетов.*

## Доступные методы решения

В зависимости от структуры графа, калькулятор использует два метода:

1. **Метод прямой подстановки (Топологический расчет)**
   * **Применение:** Только для строгих ацикличных графов.
   * **Логика:** Алгоритм Кана (топологическая сортировка). Идет строго от входа к выходу, последовательно накапливая сумму произведений входящих дуг и значений предыдущих вершин.

2. **Метод последовательного исключения вершин (Редукция графа)**
   * **Применение:** Универсальный метод (подходит как для простых, так и для графов с циклами/петлями).
   * **Логика:** Поэтапное "схлопывание" (удаление) промежуточных узлов с пересчетом весов обходных дуг (поиск путей `Вход -> Удаляемая вершина -> Выход`). Включает обработку деления на знаменатель при наличии петель `(1 - вес_петли)`. Состоит из прямого хода (редукции) и обратного хода (подстановки найденных значений).
